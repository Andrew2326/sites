# Andrework - dark tech version

Versione React ispirata al mood visivo di una landing IoT dark premium, adattata ai servizi di disegni tecnici.

## Comandi
```bash
npm install
npm run dev
npm run build
```

## Dove modificare
Apri `src/App.jsx` e cambia:
- `googleFormUrl`
- `emailAddress`
- `whatsappUrl`
- testi delle sezioni

## Pubblicazione
Usa `npm run build` e poi pubblica la cartella `dist` su GitHub Pages.
